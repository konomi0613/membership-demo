# コンポーネント化のルール

## 基本方針
必要になったら分割する。迷ったら分割しない。

---

## コンポーネント化する

### 1. 再利用する
```tsx
// ✅ 複数箇所で使う
```

### 2. 100行超えた
ファイルが長くなってきたら分割を検討

### 3. 明らかに独立している
ヘッダー、フッター、モーダルなど

---

## コンポーネント化しない

### 1. 1箇所でしか使わない
再利用しないなら page.tsx に直接書く

### 2. 100行以下
適度な長さなら分割不要

### 3. 親と密結合
propsが多すぎる場合は無理に分割しない

---

## 迷ったら
```
再利用する → コンポーネント化
再利用しない → そのまま
長くなってきた → 分割検討
```

---

## 配置場所
```
複数ページで使う → _components/
会員ページ内で使う → (member)/_components/
ページ固有 → dashboard/_components/
```
